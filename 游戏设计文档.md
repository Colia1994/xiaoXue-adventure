# 小雪闯上海 - 游戏设计文档

## 游戏概述

《小雪闯上海》是一款以雪纳瑞犬"小雪"为主角的卡牌Roguelike游戏。玩家扮演一只偷偷溜出家门的可爱雪纳瑞，在上海街头冒险，遭遇各种坏猫咪、恶霸犬和不法分子，通过卡牌战斗系统击败敌人，最终安全回家。

## 核心设计理念

### 主题设定

游戏以上海城市为背景，讲述一只名叫小雪的雪纳瑞犬的冒险故事。小雪在主人不在家时偷偷溜出门，在上海街头遇到了各种敌人：坏猫咪、凶恶泰迪、流浪大橘、城管大叔、恶霸犬、小混混，以及最终的Boss捕狗大队队长。

游戏采用轻松幽默的叙事风格，所有战斗都以狗狗的行为方式呈现，如爪击、扑咬、汪汪大叫、摇尾巴等，避免了暴力元素，保持了适合全年龄段的游戏体验。

### 游戏类型

这是一款单局制的Roguelike卡牌游戏，具有以下特点：

- 每局游戏包含6个关卡，难度逐步提升
- 卡牌带有随机基因系统，提供丰富的Build可能性
- 基因组合触发突变（组合技），增加策略深度
- 敌人具有特殊技能，需要针对性应对
- 永久死亡机制，失败后重新开始

## 核心玩法循环

### 单回合流程

1. 玩家抽5张牌，获得3点能量
2. 拖拽或点击卡牌进行攻击、防御或回血
3. 每张卡牌消耗1点能量
4. 使用完毕后点击"回合结束"
5. 敌人根据意图进行攻击
6. 进入下一回合，手牌回收重新抽牌

### 关卡推进

1. 击败当前关卡所有敌人后进入"传染阶段"
2. 随机选择一张手牌，将其基因传染给其他手牌
3. 传染后进入下一关卡，遭遇更强的敌人
4. 重复6次后击败最终Boss，获得胜利

## 卡牌系统设计

### 卡牌类型

游戏包含四种基础卡牌类型：

**攻击类卡牌**：对单个敌人造成伤害。包括爪击（3点基础伤害，4张）、扑咬（4点基础伤害，3张）、死亡翻滚（6点基础伤害，2张）。攻击类卡牌是主要的输出手段。

**防御类卡牌**：为玩家提供护甲保护。包括抱头（3点护甲，3张）、匍匐（5点护甲，2张）、躲沙发底下（7点护甲，1张）。护甲可以抵消敌人的伤害，但每回合结束时会清零。

**回血类卡牌**：恢复玩家生命值。包括狗粮（4点回血，1张）、牛皮饼干（7点回血，1张）。回血类卡牌数量稀少，需要谨慎使用。

**增益类卡牌**：提供临时增益效果。目前只有磨牙棒（下次攻击+2，2张），可以叠加使用。

**技能类卡牌**：具有特殊效果。包括汪汪大叫（震慑敌人降低攻击力）、摇尾巴（迷惑敌人可能跳过回合）、嗅探（标记敌人弱点增加下回合伤害）、撒娇（回血并抽牌）、标记领地（对所有敌人造成少量伤害并标记）。

### 基因系统

基因是卡牌系统的核心创新点。每张卡牌可能携带0-1个基因，约30%的卡牌会自带基因。基因共有8种：

- 利齿：增加2点伤害
- 硬毛：增加3点护甲
- 疾跑：先攻并冻结敌人1回合
- 嗅探：标记敌人弱点，下回合伤害翻倍
- 卖萌：回复造成伤害50%的生命
- 吠叫：伤害弹射到随机敌人
- 零食：回合结束额外抽1张牌
- 忠诚：卡牌效果翻倍

基因可以叠加，一张卡牌通过传染系统最多可以携带多个基因，形成强大的组合效果。

### 突变系统（组合技）

当一张卡牌携带两个特定基因组合时，会触发突变效果（组合技）。目前设计了10种突变：

- 铁齿铜牙（利齿+硬毛）：10点伤害加5点护甲
- 闪电爪（利齿+疾跑）：15点伤害并冻结敌人
- 致命一击（嗅探+利齿）：20点无视护甲伤害
- 治愈之吻（卖萌+忠诚）：回复15点生命
- 狮吼功（吠叫+忠诚）：全体8点伤害
- 寻味追踪（零食+嗅探）：抽3张牌
- 幽灵犬（疾跑+嗅探）：闪避下回合攻击
- 铜墙铁壁（硬毛+忠诚）：15点护甲
- 狂吠乱咬（利齿+吠叫）：随机攻击3次
- 大餐时间（卖萌+零食）：回10点生命抽2张牌

突变系统是游戏的核心策略点，玩家需要通过传染系统凑出想要的基因组合。

### 传染系统

传染系统在每击败一个敌人后触发。玩家选择一张手牌作为"传染源"，系统随机选择另一张手牌作为"传染目标"，将传染源的一个随机基因复制给目标卡牌。

传染系统是Roguelike元素的核心体现，通过单局游戏内的卡牌进化，让玩家体验到Build成型的快感。合理选择传染源和目标，可以快速凑出强力的突变组合。

## 敌人系统设计

### 敌人类型

游戏设计了7种敌人，分布在6个关卡中：

**第1关 - 坏猫咪**：15点生命，3点攻击，无护甲。技能"猫爪三连"有40%概率触发，连续攻击3次。作为新手关的敌人，帮助玩家熟悉战斗系统。

**第2关 - 凶恶泰迪**：20点生命，4点攻击，2点护甲。技能"狂吠震慑"有35%概率触发，造成伤害并降低小雪攻击力。护甲增加了战斗难度，需要玩家考虑破甲策略。

**第3关 - 流浪大橘（双体）**：两只12点生命、3点攻击的敌人。群体关卡，考验玩家的AOE能力和目标选择策略。

**第4关 - 城管大叔**：28点生命，5点攻击，3点护甲。技能"网兜抓捕"有30%概率触发，高血量高护甲的单体Boss。

**第5关 - 恶霸犬+小混混**：恶霸犬18点生命5点攻击，技能"撕咬"造成流血；小混混10点生命2点攻击，技能"扔石头"远程攻击。双体Boss战，需要同时应对两个敌人。

**第6关 - 捕狗大队队长（最终Boss）**：45点生命，7点攻击，4点护甲。技能"终极抓捕"有35%概率触发，造成10点超高伤害。作为最终Boss，具有最高的数值和压迫感。

### 敌人AI系统

每个Boss都有概率使用特殊技能而非普通攻击。技能触发概率在30%-40%之间，通过BOSS_SKILLS配置中的chance属性控制。

敌人在每回合结束时会生成下回合的意图（普通攻击或特殊技能），并在tooltip中显示给玩家，让玩家有预判和应对的机会。

部分敌人状态会影响其行为：
- 冰冻状态：敌人本回合无法行动
- 迷惑状态：50%概率跳过本回合
- 虚弱状态：攻击力降低2点
- 暴露状态：受到额外3点伤害
- 标记状态：被标记的敌人会受到持续伤害

### Boss技能设计

Boss技能分为多种类型：

- 多重攻击（multi）：连续攻击多次，如猫爪三连
- 重击（heavy）：单次高伤害攻击，如肥猫压顶
- 流血（bleed）：伤害加持续流血效果，如撕咬
- 终极技能（ultimate）：超高伤害，如终极抓捕

技能设计体现了每个敌人的特色，如猫咪的连续爪击、泰迪的狂吠、大橘的体重压制等，符合角色的设定。

## 玩家角色系统

### 基础属性

小雪（玩家角色）的基础属性：
- 生命值：40点
- 每回合能量：3点
- 护甲：动态值，每回合清零

### 状态系统

小雪可以进入多种状态：
- 攻击动画：使用攻击卡牌时触发
- 防御动画：使用防御卡牌时触发
- 回血动画：使用回血卡牌时触发
- 增益状态：使用磨牙棒后攻击力提升

## 技术实现要点

### 前端技术栈

游戏使用以下技术栈开发：
- React 18：核心框架，使用函数组件和Hooks
- Vite：构建工具，提供快速开发体验
- CSS3：动画和视觉效果
- Web Audio API：8bit风格音效系统

### 核心系统实现

**卡牌拖拽系统**：实现了完整的拖拽交互，包括鼠标按下检测、拖拽阈值判断、目标位置计算、插入动画等。支持点击选择+目标选择的备用交互方式。

**战斗计算系统**：采用函数式计算，根据卡牌基础属性、基因加成、突变效果、玩家增益等综合计算最终效果。伤害计算考虑护甲减免、暴击、穿透等因素。

**敌人AI系统**：基于意图的AI设计，每回合结束时预测下回合行动（普通攻击或技能），通过概率系统控制技能触发频率。

**传染系统**：在战斗结束后触发，随机选择基因进行复制，实现卡牌进化。使用基因池和组合检测算法。

**音效系统**：使用Web Audio API动态生成8bit风格音效，包括方波、锯齿波、正弦波等波形合成，以及扫频、包络等技术。为每种卡牌和Boss技能设计了独特音效。

**动画系统**：使用CSS Keyframe动画实现卡牌悬浮、敌人浮动、攻击动作、受击反馈等效果。使用transform和opacity实现硬件加速。

### 响应式设计

游戏采用响应式布局，适配桌面端和移动端：
- 使用clamp()函数实现字体和尺寸的平滑缩放
- 使用vw单位实现基于视口宽度的自适应
- 移动端优化了触摸交互和UI布局
- 卡牌区域、敌人区域、日志区域采用弹性布局自动调整

### 性能优化

**虚拟DOM优化**：使用React的key属性正确标识列表项，避免不必要的重渲染。

**状态管理优化**：使用useRef同步状态，避免闭包陷阱。使用函数式更新避免状态竞争。

**动画性能**：使用transform和opacity实现GPU加速，避免触发布局重排。

**音效优化**：使用单例AudioContext，避免重复创建。音效播放使用低延迟的振荡器直接合成。

## 游戏平衡设计

### 数值平衡

**卡牌数值**：攻击类卡牌基础伤害3-6点，防御类3-7点护甲，回血类4-7点。通过基因和突变可以显著提升效果，但需要投入传染机会。

**敌人数值**：从第1关到第6关，敌人生命值从15点递增到45点，攻击力从3点递增到7点，护甲从0递增到4点。难度曲线平滑，给玩家成长空间。

**资源管理**：每回合3点能量限制，手牌5张上限，迫使玩家做出取舍。回血卡牌数量稀少（仅2张），增加生存压力。

### 策略深度

游戏提供了多层策略选择：

1. **即时决策**：每回合选择使用哪些卡牌，优先攻击哪个敌人，是否使用防御或回血。

2. **Build规划**：通过传染系统凑基因组合，决定本局游戏的核心战术（如冰冻流、吸血流、AOE流等）。

3. **风险评估**：是否冒险不防御以保留输出，是否使用回血卡牌还是留着应对后续关卡。

4. **目标选择**：面对多个敌人时，优先击杀哪个目标，是否使用AOE技能。

## 未来扩展方向

### 内容扩展

- 新增更多卡牌类型和技能效果
- 增加更多敌人和关卡，延长游戏流程
- 添加稀有卡牌或特殊事件系统
- 增加成就系统和收集要素

### 系统扩展

- 添加难度选择（简单、普通、困难）
- 实现存档系统，支持中途退出继续游戏
- 添加数据统计（胜率、最高伤害、使用最多的卡牌等）
- 实现每日挑战模式（固定种子）

### 视觉和音效

- 为每张卡牌设计独特插画
- 增加更多敌人动画和受击反馈
- 添加背景音乐的不同曲目
- 实现音效开关和音量调节

## 总结

《小雪闯上海》是一款融合了Roguelike元素的卡牌策略游戏。通过基因系统和传染机制，在传统的卡牌战斗基础上增加了Build构筑的乐趣。轻松幽默的主题和可爱的角色设计，让游戏具有亲和力，而深度的策略系统则提供了足够的挑战性和重复游玩价值。

游戏的核心循环简洁明了：战斗、传染进化、再战斗。通过6个关卡的递进难度，玩家需要在有限的资源下做出最优决策，同时通过传染系统不断强化自己的卡牌，最终击败强大的Boss。

技术实现上，游戏充分利用了React的组件化思想和现代Web技术，实现了流畅的交互体验和视觉效果。响应式设计确保了在不同设备上的良好体验，为后续可能的平台扩展打下了基础。
